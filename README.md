# О проекте

Rescue AI — это автономная система для обнаружения людей на видеопотоке БПЛА в сложных условиях без облачной поддержки и GPS с алгоритмом восстановления траектории дрона, обеспечивающая высокую точность детекции человека в реальном времени и применимая в реальных поисково-спасательных операциях, включая стихийные бедствия, горы и леса

## Основные функции

- Обработка видеопотока: подключение к источнику (RTSP или видеофайл), чтение кадров, стабильная работа при обрывах/задержках.
- Компьютерное зрение: детекция объектов/людей на кадре, базовый пост‑процессинг и агрегирование результатов по времени.
- Навигационная/управляющая логика: 2D/3D-визуализация полёта, расчёт состояния, траектории и подсказок на основе наблюдений из видео (и доп. сигналов, если появятся).
- Интерфейс/интеграция: удобный способ запускать обработку, видеть прогресс и результаты (сводка, графики/телеметрия), получать отчёт по сессии.
