# О проекте

Rescue AI — это система для обнаружения людей на видеопотоке БПЛА в сложных условиях без облачной поддержки и GPS, обеспечивающая высокую точность детекции человека в реальном времени и применимая в реальных поисково-спасательных операциях, включая стихийные бедствия, горы и леса.


## [**ML System Design Doc**](docs/ml_system_design_doc.md)

## Основные функции

- Обработка видеопотока: подключение к источнику, чтение кадров, стабильная работа при обрывах/задержках.
- Компьютерное зрение: детекция объектов/людей на кадре, базовый пост‑процессинг и агрегирование результатов по времени.
- Интерфейс/интеграция: удобный способ запускать обработку, видеть прогресс и результаты (сводка, графики/телеметрия), получать отчёт по сессии.

## Запуск текущей версии

Сейчас в репозитории поднят минимальный API с эндпоинтом проверки состояния `GET /health`.

### Вариант 1: через Docker

1. В корне проекта должен быть файл `.env` (можно пустой для текущего этапа).
2. Запуск:

```bash
docker compose up --build
```

3. Проверка:

```bash
curl http://127.0.0.1:8000/health
```

Ожидаемый ответ:

```json
{"status":"ok"}
```

### Вариант 2: локально без Docker

```bash
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python -m uvicorn services.api_gateway.app:app --host 0.0.0.0 --port 8000
```

Проверка та же:

```bash
curl http://127.0.0.1:8000/health
```
