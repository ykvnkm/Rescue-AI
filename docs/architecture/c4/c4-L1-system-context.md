## Описание
На уровне L1 показываем систему как “чёрный ящик”: кто с ней взаимодействует и какие внешние зависимости есть у решения.

## Диаграмма (L1)

```mermaid
flowchart LR
  OP["Оператор ДПСО"] -->|UI / REST| SYS["Rescue-AI: детекция людей, алерты, отчёт миссии"]
  SYS -->|алерты / отчёт / ключевые кадры| OP

  SRC["Источник видео/кадров: офлайн папка или стрим"] -->|кадры| SYS

  EXT1["Файлы миссии: кадры/аннотации (MISSION_DIR)"] --- SYS
  EXT2["Runtime-директория: кэш/артефакты (./runtime)"] --- SYS

  %% Production/план
  DB["Postgres (планируется)"] -.-> SYS
  S3["S3-хранилище (планируется)"] -.-> SYS
  MON["Мониторинг Prometheus/Grafana (планируется)"] -.-> SYS
```
